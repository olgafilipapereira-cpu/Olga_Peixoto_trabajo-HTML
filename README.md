# Proyecto final HTML/CSS

Sitio web ficticio desarrollado con HTML5 y CSS3 como ejercicio final del módulo.

## Estructura del proyecto

- `index.html`: portada con tres secciones
- `portfolio.html`: galería de trabajos en cuadrícula
- `presupuesto.html`: formulario con validación HTML5
- `contacto.html`: datos de la empresa y mapa ilustrado
- `assets/css/styles.css`: hoja de estilos principal
- `assets/img/`: recursos gráficos del sitio

El proyecto utiliza una estructura de carpetas simple y ordenada, comentarios en el código y una paleta de color limitada para mantener coherencia visual.
